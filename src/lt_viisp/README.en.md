Automates authentication on Lithuanian government portals. Removes the need to click through multiple bank selection pages.

### Features
- Adds a direct "Auth via SB" button to skip multiple selection pages.
- Supported sites: [Sodra](https://www.sodra.lt), [Migris](https://www.migracija.lt), [Esveikata](https://www.esveikata.lt).
- Smart-ID input on the bank page remembers your ID.
