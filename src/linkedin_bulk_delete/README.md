Массовое удаление уведомлений в LinkedIn.  
Добавляет чекбоксы для выбора нескольких уведомлений и их удаления.

### Функционал
 Добавляет чекбоксы выбора слева от уведомлений.
 Удаление выбранных или всех видимых уведомлений одной кнопкой.
 Работает без перезагрузки страницы (использует встроенную логику LinkedIn).
